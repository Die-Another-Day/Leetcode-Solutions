## About

This repository is a **personal commitment** to solving **one LeetCode problem every single day**. Each solution is crafted with three core principles:

| Principle     | Description                                     |
|---------------|-------------------------------------------------|
| **Correctness** | Passes 100% of LeetCode test cases            |
| **Efficiency**  | Optimized for time and space complexity       |
| **Readability** | Clean, well-structured, self-documenting code |

> *"Consistency over intensity — small daily progress compounds into mastery."*

## Problem Format

Each problem folder contains exactly **two files**:

### 1. `Solution.java`

A self-contained, compilable solution class — ready to copy and submit on LeetCode.

```java  
class Solution {  
    public returnType methodName(params) {  
        // Efficient implementation  
    }  
}  
```

### 2. `README.md` (inside problem folder)

A concise, structured explanation of the approach:

```markdown  
# Problem Title  

**Difficulty:** Easy / Medium / Hard  

## Approach  
- High-level strategy  
- Key insight that drives the solution  

## Complexity  
- **Time:** O(...)  
- **Space:** O(...)  

## Algorithm  
1. Step one  
2. Step two  
3. Step three  
```
